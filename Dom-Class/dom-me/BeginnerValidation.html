Simple Form Validation for Beginners

What we're learning: Basic form validation using JavaScript to check if users enter correct information.
Please enter your name
Please enter a valid email
Age must be 18 or older
Great! All information is correct. Form submitted successfully!

How This Works:

  • Name: Must not be empty
  • Email: Must contain @ and a domain (like .com)
  • Age: Must be a number 18 or higher

Try submitting with empty fields or wrong information to see the validation in action!
